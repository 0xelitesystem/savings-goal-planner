# Savings Goal Planner

> Solve for the monthly amount, or for how long the goal takes.

**Live demo:** https://0xelitesystem.github.io/savings-goal-planner/

Not financial advice. This is an estimate from the numbers you enter, for learning and planning only.

Single HTML file. Runs in the browser with no build step, no server, no tracking, and no data leaving the page.

## What it does

- Mode one: enter a target and a deadline to get the monthly contribution needed
- Mode two: enter a monthly contribution to get the time to reach the goal
- Optional annual return applies monthly compounding to the balance and contributions
- A thermometer shows how much of the goal is already saved

## What it is not

- Not financial advice and not a guarantee. Returns are assumed, never promised
- Not a budgeting app. It does not track accounts or store anything

## Use it

Open the hosted page: https://0xelitesystem.github.io/savings-goal-planner/

Or download `index.html` and open it in any browser. It works offline.

## Privacy

Everything runs client-side. No analytics, no cookies, no network calls, no local storage.
